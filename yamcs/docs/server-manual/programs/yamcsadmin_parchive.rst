yamcsadmin parchive
===================

.. program:: yamcsadmin parchive

**NAME**

    yamcsadmin parchive - Parameter Archive operations


**SYNOPSIS**

    ``yamcsadmin parchive --instance INSTANCE COMMAND``


**OPTIONS**

    .. option:: --instance INSTANCE

        Yamcs instance.


**COMMANDS**

    :doc:`print-pid <yamcsadmin_parchive_print-pid>`
        Print parameter name to parameter id mapping.
    :doc:`print-pgid <yamcsadmin_parchive_print-pgid>`
        Print parameter group compositions.
